### 実行方法

1. firebase の秘密鍵を取得する
2. env に秘密鍵を設定する
3.

```bash
export GOOGLE_APPLICATION_CREDENTIALS=秘密鍵のパス
echo $GOOGLE_APPLICATION_CREDENTIALS
```


4. yarn start:dev
